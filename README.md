# Chuck Norris Jokes

This CLI app gets a joke from the Chuck Norris API after taking a category of your preference as an input.

# Tech Stack

-Javascript -Node.js -API

## Features

- Uses Node FileSystem
- Fetched data from an API
- Styles through Chalk
